# bisca-je

An implementation of both [Bisca](https://pt.wikipedia.org/wiki/Bisca) and [Briscola](https://en.wikipedia.org/wiki/Briscola).

## References

* https://pt.wikipedia.org/wiki/Bisca

* https://en.wikipedia.org/wiki/Briscola

## How to run:

```
$ npm install

$ webpack install

$ npm run lite

```
