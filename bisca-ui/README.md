# BiscaJE UI

## Building the game:

*  Node js legacy (not necessary!)
```
# Node JS for LINUX
$ sudo apt install nodejs-legacy 
# OR
# Node JS for MACOS
$ sudo brew install nodejs-legacy
```

* Common installation:

```
$ npm install

$ tsc

```

* Running server options:
```
# Choose one:

$ webpack install

$ npm run lite

# or

$ npm run front

```
